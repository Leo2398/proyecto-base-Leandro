Cómo instalar

Sigue estos pasos para evitar errores al abrir la aplicación en Android o Windows.

Importante antes de usar

AgroMarket necesita conexión a internet para comunicarse con la base de datos y cargar usuarios, productos, pedidos, recargas y reportes. En Windows no abras el ejecutable dentro del ZIP: primero debes extraer todo.

1

Descargar Android

Presiona el botón del APK, descarga el archivo e instálalo en tu dispositivo Android.

2

Permitir instalación

Si Android lo solicita, permite instalar aplicaciones desde el navegador o gestor de archivos.

3

Descargar Windows

Descarga el ZIP de Windows y extrae la carpeta completa en tu computadora.

4

Abrir el .exe

Después de extraer, abre el archivo ejecutable de AgroMarket desde la carpeta extraída.
